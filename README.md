# /coords {player}

A simple mod that adds a command that broadcasts the coordnates of a player in chat.

Requires [Architectury API](https://modrinth.com/mod/architectury-api)

## Usage:

To brodcast your own coordnates:
```
/coords
```

To brodcast someone elses coordnates:
```
/coords {player}
```

To toggle the mod on/off (op required)
```
/coords-toggle
```
